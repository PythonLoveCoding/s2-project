# s2-project
## Global Hotkeys
(Works even when the overlay is locked)

**F8**  
Toggle Click-through (Lock / Unlock overlay)

---

## Overlay Controls

**Left Click**  
Select & move pieces

**Right Click + Drag**  
Move overlay window

**F**  
Flip board

**Ctrl + Z**  
Undo move

**Ctrl + R**  
Reset board

---

## Board & Engine

**+ / -**  
Increase / decrease board size

**[ / ]**  
Decrease / increase engine Elo

**0**  
Cycle depth (5 → 10 → 15 → 20)

---

## System

**ESC**  
Quit application
